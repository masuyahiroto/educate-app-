# Educate 管理者用取扱説明書
## 〜家庭教師の先生向け〜

---

## はじめに

**Educate** は家庭教師の授業管理を効率化するWebアプリです。生徒ごとの宿題・ワーク進捗・テスト・授業日程を一元管理できます。

### アクセス方法
- ブラウザで `http://localhost:5173` を開く
- スマートフォン・パソコンどちらでも利用可能
- データはこのパソコンのブラウザに自動保存されます（インターネット不要）

> ⚠️ **注意：** ブラウザのキャッシュ・履歴を「すべて削除」するとデータが消えます。定期的にバックアップを取ることをお勧めします。

---

## 画面の見方

左側（スマホは左上の「≡」ボタン）にナビゲーションメニューがあります。

| メニュー | 説明 |
|---|---|
| ダッシュボード | 全体の概況を確認する画面 |
| 生徒管理 | 生徒の登録・編集・削除 |
| 宿題管理 | 宿題の出題・完了確認 |
| ワーク進捗 | 問題集のページごとの進捗管理 |
| カレンダー | 授業日・提出日・テスト日の一覧 |
| テスト | テストの作成・採点・記録 |

## 1. ダッシュボード

アプリを開いたときに最初に表示される画面です。

### 表示される情報
- **生徒数** ：登録している生徒の合計人数
- **未完了の宿題** ：まだ完了していない宿題の件数
- **今日の授業** ：当日に設定した授業の数
- **予定テスト数** ：今日以降に予定されているテストの数

### 下部の一覧
- **今日の授業** ：当日の授業時間と対象生徒
- **未完了の宿題** ：期限が近い・超過した宿題（赤＝期限超過、黄＝期限あり）
- **予定テスト** ：直近3件のテスト予定
- **生徒一覧** ：各生徒の未完了宿題件数

## 2. 生徒管理

### 生徒を追加する
1. 右上の「**生徒を追加**」ボタンをクリック
2. 以下の情報を入力：
   - **名前**（必須）：生徒の氏名
   - **学年**：小1〜高3から選択
   - **担当科目**：国語・数学・英語・理科・社会から複数選択可（ワーク・宿題では「その他」も選択可）
   - **メモ**：特記事項（苦手分野、連絡事項など）
3. 「保存」ボタンをクリック
4. 生徒カードに**アクセスキー**（例：`A4F2K9`）が自動発行されます

### 生徒情報を編集する
- 生徒カードをクリックすると編集画面が開きます

### 生徒を削除する
- 生徒カード右上の🗑️アイコンをクリック
- （削除すると元に戻せません）

---

## 生徒用URLの発行と共有

生徒ごとに専用のアクセスURLを発行できます。このURLを生徒に伝えると、生徒が自分のスマホやパソコンから自分のデータだけを確認・操作できます。

### URLの仕組み

```
https://（サイトのアドレス）/?student=A4F2K9
```

- `A4F2K9` の部分が**アクセスキー**（生徒ごとに異なる6桁の英数字）
- URLを知っている人だけがアクセスできます（パスワード不要）
- 他の生徒のデータは見えません

### URLの確認・コピー方法

1. 「**生徒管理**」画面を開く
2. 生徒カードの下部にアクセスキーが表示されています
3. 「**URLをコピー**」ボタンをクリックするとURLがクリップボードに入ります
4. LINEやメールなどで生徒に送ってください

> 📌 **既存の生徒にキーがない場合**：カード下部の「URLを発行」ボタンをクリックするとキーが生成されます。

### 生徒画面でできること

| 操作 | 可否 |
|---|---|
| 宿題の完了・未完了を切り替える | ✅ |
| ワークのページを完了・未完了にする | ✅ |
| 今後の予定（授業・宿題・テスト）を確認する | ✅ |
| テストの結果・問題を確認する | ✅ |
| 宿題・ワーク・テストを追加・削除する | ❌（先生のみ） |

### セキュリティについて

- URLを第三者に知られると、その生徒のデータが見られてしまいます
- 流出が心配な場合は「URLを発行」ボタンで新しいキーを再発行してください（古いURLは無効になります）

## 3. 宿題管理

### 宿題を出す
1. 右上の「**宿題を追加**」ボタンをクリック
2. 以下を入力：
   - **生徒**（必須）：対象の生徒を選択
   - **科目**：国語・数学・英語・理科・社会・その他から選択
   - **内容**（必須）：宿題の内容（例：教科書 p.42〜45 の問題）
   - **提出期限**：次の授業日などを設定
   - **メモ**：補足事項
3. 「追加」をクリック

### 宿題の完了を確認する
- 一覧の左端の○ボタンをクリックで完了/未完了を切り替えます
- ✔（緑）＝完了、○（グレー）＝未完了

### 表示フィルター
- 「すべて」「未完了」「完了済み」で絞り込みができます

### 宿題一覧の列
| 列名 | 内容 |
|---|---|
| 完了 | クリックで完了/未完了を切り替え |
| 生徒 | 対象の生徒名 |
| 科目 | 選択した科目 |
| 内容 | 宿題の内容 |
| 提出期限 | 設定した期日 |
| 出題日 | 宿題を追加した日付（自動記録） |

### 宿題を削除する
- 各行右端の🗑️アイコンをクリックすると削除できます
- 削除したデータは復元できません

### 期限の色の意味
- 🔴 **赤** ：期限を超過している
- 🟡 **黄** ：期限が設定されている（まだ余裕あり）

---

## 4. ワーク進捗管理

問題集（ワーク）のページごとの進捗を視覚的に管理します。

### ワークを登録する
1. 「**ワークを追加**」ボタンをクリック
2. 以下を入力：
   - **生徒**（必須）
   - **科目**：国語・数学・英語・理科・社会・その他から選択
   - **ワーク名**（必須）：例「中学数学 標準問題集」
   - **開始ページ**：ワークの最初のページ番号
   - **総ページ数**：ワーク全体のページ数
3. 「追加」をクリック

### ページの進捗を記録する
- 各ページのマスをクリックするだけで完了（緑）/未完了（グレー）を切り替えられます
- 授業で終わったページをその場でクリックして記録してください

### 進捗バーの色の意味
- 🔴 **赤** ：進捗0〜30%（ほとんど未着手）
- 🟡 **黄** ：進捗30〜70%（進行中）
- 🟢 **緑** ：進捗70〜100%（ほぼ完了）

### ワークを削除する
- ワークカード右上の🗑️アイコンをクリックすると削除できます
- 削除するとページ進捗データもすべて消えます

### 生徒フィルター
- 上部のボタンで特定の生徒のワークだけ表示できます

## 5. カレンダー

### 授業日を登録する
1. 「**授業日を追加**」ボタンをクリック
2. 以下を入力：
   - **生徒**（必須）
   - **日付**
   - **開始時刻・終了時刻**
   - **メモ**
3. 「追加」をクリック

### カレンダーの表示内容
カレンダーの各日付に以下がカラーラベルで表示されます：

| 色 | 意味 |
|---|---|
| 🟡 黄 | 宿題の提出期限 |
| 🔵 青 | 授業日 |
| 🔴 赤 | テスト予定日 |

### カレンダーの操作
- 左右の「<」「>」ボタンで月を移動できます
- 右側の「今後の予定」リストで直近10件の予定を一覧確認できます
- 「授業日の一覧」には最新8件の授業日が表示されます

### 授業日を削除する
- 右側「授業日の一覧」から🗑️アイコンをクリック

---

## 6. テスト

### テストを作成する
1. 「**テストを作成**」ボタンをクリック
2. 基本情報を入力：
   - **生徒**（必須）
   - **科目**
   - **テストタイトル**（必須）：例「二次方程式 確認テスト」
   - **実施予定日**
3. 「**問題を追加**」ボタンで問題を作成：

| 問題形式 | 説明 |
|---|---|
| 記述 | 自由記述。模範解答を設定できる |
| 選択 | 4択。正解の選択肢にラジオボタンを選ぶ |
| ○× | ○か×か。正解を選ぶ |

4. 各問題に配点（点数）を設定
5. 「作成」をクリック

### 採点する
1. テスト一覧の「**採点**」ボタンをクリック
2. 問題形式ごとに以下の方法で解答を入力：

| 形式 | 採点方法 |
|---|---|
| 選択 | 生徒が選んだ選択肢をラジオボタンで選択（正解には「✓」マーク表示） |
| ○× | 生徒の解答（○か×）をラジオボタンで選択 |
| 記述 | 模範解答を参照し「正解」「不正解」をドロップダウンで選択 |

3. 「採点を保存」をクリック
4. テスト一覧に合計スコアが表示されます（例：38 / 50点）

### テストの詳細を見る
- テスト名左の「∨」ボタンで問題一覧を展開して確認できます

---

## よくある操作

### データが消えてしまった場合
- ブラウザのキャッシュを削除した場合はデータが復元できません
- 大切なデータは定期的にメモなどに記録しておくことをお勧めします

### スマートフォンで使う場合
- 左上の「≡」ボタンをタップしてメニューを開きます
- 操作はパソコンと同じです
